Privacy Policy

Substack Notes Scheduler is a Chrome extension that helps users draft, schedule, and manage Substack Notes directly from their browser.

Data Collection

This extension does not collect, transmit, or store any personal data or user content outside of your local browser. All notes, drafts, and settings are stored locally using Chrome's storage API.

Permissions

The extension requests only the permissions necessary to provide its features, such as storage, alarms, and access to substack.com for posting notes.

Third Parties

No data is sold, shared, or transferred to third parties.

Contact

If you have any questions about this privacy policy, please contact the developer via the Chrome Web Store listing.
